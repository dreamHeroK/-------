## 一、技能体系说明

1. **技能获取途径**：宠物技能俗称"兽决"，通过打书（书铁）、领悟、洗炼、内丹、进阶特性等系统获得，携带在宠物的技能列表中，战斗开始时统一由 `添加技能属性` 结算生效。
2. **等级替代关系**：多数技能分"普通 → 高级 → 超级"三档，**同名技能只有最高档生效，低档效果不叠加**（代码中通过 `取技能重复` 判断，低档技能若检测到宠物同时拥有更高档同名技能则直接跳过赋值）。
   - 普通、高级两档技能可通过打书/兽决书直接习得（对应技能表中的具体条目）。
   - 超级档技能不是独立的书铁条目，而是通过额外的"超级技能"养成体系（代码中 `Superjnsuoyou` 判定表 + "超级否定信仰"等特殊机制）叠加解锁，本文中随普通/高级一并列出对应数值。
3. **技能来源限定**：部分技能（如"北冥之渊""水击三千""扶摇万里""逍遥游"等）只有特定超级宠物模型天生携带，无法通过普通打书获得，详见第五节"天生技能对照表"。

### 3.5 超级否定信仰的"超级赐福"状态（元吉～万利，含"超级五福"）

"超级否定信仰"是一个特殊的超级技能：宠物同时携带它和其他超级技能时，其它所有超级技能会被**吞并失效**（效果被移除或降级为普通/高级数值，详见 `战斗处理.lua` → `添加超级否定信仰属性` 第3646行起），只保留"超级否定信仰"自身生效，换取气血/防御等换算属性。

- **效果来源**：`添加超级否定信仰属性` 统计宠物拥有的超级技能总数`yycjjn`（含"超级否定信仰"本身），据此调整：
  - 最大气血、当前气血：各 +（超级技能数量×20%）
  - 防御、法防：固定 -5%
  - 清除神佑复生/鬼魂术/感知/四大元素吸收等效果
  - 其余每个超级技能被标记为"超否失效_技能名"，效果按代码逐条置空或降回普通/高级档数值（如超级遗志→遗志2档、超级进击必杀→进击必杀30等）
- **"超级赐福·X"只是一个纯展示用的战斗状态图标**，无独立数值效果（`技能数据库.lua` 中 `状态属性` 一律只返回 `{类型=2,状态回合=150}`，仅代表持续150回合的图标）。战斗中每回合会按宠物当前拥有的超级技能数量`超级否定信仰`字段（即`超级技能数量`，见`战斗处理.lua`第5652行"添加技能属性"）去查表 `超级技能状态名称[数量]` 取一个对应称号显示出来：

| 超级技能数量 | 赐福称号 |
|---|---|
| 1 | 超级赐福·元吉 |
| 2 | 超级赐福·双喜 |
| 3 | 超级赐福·三合 |
| 4 | 超级赐福·四季 |
| 5 | 超级赐福·五福（俗称"超级五福"） |
| 6 | 超级赐福·六临 |
| 7 | 超级赐福·七神 |
| 8 | 超级赐福·八欲 |
| 9 | 超级赐福·九功 |
| 10 | 超级赐福·十满 |
| 11及以上 | 超级赐福·万利 |

> 数量越多只代表宠物身上被"否定信仰"吞并的超级技能越多（称号越高），**称号本身不额外加成**，实际收益只来自上面"效果来源"里的固定公式（气血/防御换算 + 其余超级技能失效降档）。若同时拥有"超级否定信仰"和 N 个其他超级技能，收益等价于"1个超级否定信仰基础效果 + N×20%气血加成"，其余那 N 个技能则各退化为其普通/高级版本效果（或直接失效，视具体技能而定）。
