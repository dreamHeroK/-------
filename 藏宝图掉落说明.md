# 藏宝图类道具（藏宝图/高级藏宝图/玲珑宝图）产出与开图掉落说明

> 本文档梳理【藏宝图】【高级藏宝图】【玲珑宝图】三种"挖宝"类道具的获取途径、使用限制，以及使用后各自的随机开图掉落表。核心逻辑集中在 `Script/角色处理类/道具处理类.lua`（使用入口 6431-6450 行，地图/坐标生成 9845-9852 行，三个"处理"函数 12641-12803 行）。

## 一、道具总览

| 道具   | 使用后调用的处理函数    | 函数所在行           |
| ---- | ------------- | --------------- |
| 藏宝图  | `低级藏宝图处理(id)` | 道具处理类.lua:12736 |
| 高级藏宝图 | `高级藏宝图处理(id)` | 道具处理类.lua:12641 |
| 玲珑宝图 | `玲珑宝图处理(id)`  | 道具处理类.lua:12765 |

**共同的使用限制**（道具处理类.lua:6431-6450）：

- 必须是**队长**（`取队长权限`），否则提示"你不是队长，无法使用此道具！"。
- 道具生成时（给予道具那一刻，道具处理类.lua:9845-9852）会从下列11张地图中随机绑定一张地图+随机坐标：

  建邺城(1501)、东海湾(1506)、傲来国(1092)、长寿郊外(1091)、大唐国境(1110)、女儿村(1142)、花果山(1514)、北俱芦洲(1174)、大唐境外(1173)、五庄观(1146)、朱紫国(1208)

- 使用时若当前地图与道具绑定地图不一致，直接**自动跳转**过去（原本按坐标距离校验"这里没有宝藏"的逻辑已被注释掉，形同虚设）。
- 三种宝图共用同一套地图/使用规则，仅"开图"后的奖励表不同。

## 二、获取途径

1. **商城购买**：`Script/商店处理类/商城处理类.lua:170`，藏宝图/高级藏宝图/玲珑宝图统一走"直接给予"分支，具体售价由商城前端配置决定，源码内未出现具体数字。
2. **翻一翻夺宝小游戏**（`Script/商店处理类/翻一翻.lua`）：奖池里**只有【高级藏宝图】**，权重10（同档位的"高级召唤兽内丹"权重5、"神兜兜"权重10）。每日有免费次数（角色.翻一番），用完后消耗仙玉继续翻，仙玉消耗和倍率随连续翻牌次数递增。
3. **"宝图任务"**（`Script/任务处理类/活动调用/Task4.lua`）：NPC"店小二"处选择"听听无妨(消耗2000两银子)"领取任务，前往地图击杀随机生成的"强盗"，胜利后固定获得3000~8000随机银子，并有**75%概率**获得1张【藏宝图】。
4. **答题类NPC奖励**（`Script/系统处理类/游戏活动类.lua` 616-631/733-747/837-851行）：灯谜老人 / 知识老人 / 房都尉三个答题活动逻辑完全相同，每答对一题在1~200里抽奖，≤20（**10%**）获得1张【藏宝图】。
5. **特赦令牌任务奖励**（道具处理类.lua:11466-11469）：玩家交出"特赦令牌"后，固定获得1张【高级藏宝图】。
6. **副本/任务战斗奖励表**（`数据表/物品奖励数据.xls`，按"战斗类型"配置，运行时由 `获取__物品奖励数据()` 加载，见 `Script/初始化脚本.lua`）：部分副本战斗会把藏宝图/高级藏宝图放进按权重抽取的物品池，例如"雁塔地宫1层boss"（战斗类型130030）权重池里【高级藏宝图】权重70（详见仓库内《怪物掉落和任务掉落.md》第五节样例2）。该表共146+行配置，本文不逐行穷举，仅提示这是另一条掉落来源。

**已确认不生效 / 死代码，供排查参考**：

- `Script/系统处理类/游戏活动类.lua` 第416行附近另有一段给【藏宝图】的分支，整段被注释掉，当前不会触发。
- `道具处理类:完成宝图遇怪(id)`（道具处理类.lua:12805）仍然存在，但唯一的调用点在 `Task4.lua:161` 也被注释掉了，属于无人调用的死代码。
- `设置任务102`（`Script/任务处理类/活动调用/Task102.lua:45`，"宝宝窝"）：原本由【高级藏宝图】66~75区间触发，现已改为发放"书铁"，改动后仓库内已无任何地方调用此函数，属于死代码。

## 四、涉及的"放怪"任务简述

三种宝图的部分掉落分支不是直接给道具，而是在地图上刷出一批怪物供全服玩家争抢，本人不保证能获得奖励：

- **设置任务205**（`Script/任务处理类/活动调用/Task205.lua:47`）：在1177/1186/1201/1203四张地图之一刷3~5只"避世老妖/冥府妖王/猪仙大圣"，需≥50级、组队≥3人才能开打（战斗ID 100020）。
- **设置任务101**（Task205.lua:125）：在1501/1193/1110/1173/1092/1091/1174/1210八张地图之一按等级刷11只"远古XX"封妖怪，最后3只为变异怪，组队≥3人才能开打（战斗ID 100004），变异怪额外奖励经验/银子。
- **设置任务102**（`Script/任务处理类/活动调用/Task102.lua:45`）：在10张地图之一刷2~4只随机模型"宝宝"，10分钟内未被击杀会自动消失。**注意**：原本【高级藏宝图】66~75区间会触发该任务，现已改为直接发放"书铁"，改动后全仓库已无任何地方调用`设置任务102`，该函数目前是死代码。
