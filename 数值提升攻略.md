# 人物与宠物数值提升攻略

> 基于服务端源码分析整理（Lua），涉及文件路径与关键函数均来自实际代码。

## 一、人物属性提升攻略

**核心文件**：`Script/角色处理类/角色处理类.lua`（属性总控 `刷新信息`，约4798行）

### 第1步：练级攒基础五维
- 打怪/任务/活动获得经验 → `添加经验()`，实际经验 = 数额 × 服务端经验倍率 × VIP加成
- 满经验触发 `升级处理()`：每级 体质/魔力/力量/耐力/敏捷 各+1，潜力+5，气血上限/活力上限各+10
- 69/89/109/129/144级是节点，会二次确认（属性质变点）

### 第2步：分配潜力点 + 用潜能果/月饼补潜力
- 潜力点自由加到五维上（力量→命中/伤害，耐力→防御，体质→气血，魔力→法力，敏捷→速度，具体系数见 `取属性()`，人/魔/仙三套系数不同）
- 找NPC老猕猴（对话 `1514.lua`）用经验换潜力点，消耗经验 = 已兑换个数×15万+1000万，随等级/渡劫解锁上限（60级50个 → 90级100个 → 155级+渡劫150个 → 170级+渡劫200个）。**潜能果本身不是实体道具，只能靠这个兑换入口获得**
- 道具"月饼"直接+1潜力，每人限100次。**来源**：中秋活动任务、中秋积分商店、幸运转盘抽奖、双龙签到、阵营积分商店（未发现仙玉商城直购）

### 第3步：帮派点修（被动战斗加成，性价比最高的早期投资）
- 攻击/防御/法术/抗法/猎术修炼，每级消耗银子3万+帮派资材5点（受帮贡上限约束）
- **帮派资材来源**：不是掉落/任务道具，而是帮派整点维护时系统自动产出（`帮派处理.lua`），上限20万，攒够了直接去帮派NPC点修即可
- 战斗中每点攻修/防修≈2%物理伤害/减伤，法术修炼/抗法修炼同理影响法伤，猎术修炼加"追加攻击"几率
- 修炼上限随等级解锁：`min(20, floor((等级-20)/5))`，所以先冲等级才能继续点修炼

### 第4步：装备成型
1. 先打造/购买基础装备（打造消耗 = 金钱公式(等级) + 体力）。**打造材料（制造指南书/百炼精铁）来源**：怪物/任务掉落表、NPC"杂货购买员"银子购买、比武积分兑换、仙玉商城直购、"搏一搏"随机礼包
2. 装备练"锻炼等级"后镶宝石（宝石等级需匹配锻炼等级）。**宝石来源**：怪物/副本/任务掉落（主渠道）、NPC"宝石商人"银子购买、抽奖活动、新手礼包（未见仙玉商城出售）
3. 高等级装备叠符石组合、灵饰淬灵/星辉石。**符石来源**：仙玉商城直购未激活符石、NPC"云游道人"银子购买符石卷轴、副本/任务掉落、活动奖励、阵营积分商店；**灵饰材料（元灵晶石/灵饰指南书）来源**：仙玉商城、普通NPC商店、三族时空积分商店、比武积分、怪物掉落、幻域迷宫副本、"搏一搏"礼包
4. 3件/5件动物套提供额外套装加成

### 第5步：师门技能与生活技能点满
- 师门技能升级消耗经验+银子（表随等级陡增），优先点主output技能
- 生活技能（强身术+1%气血、冥想+1%魔法、神速+50%速度系数等）性价比高，建议早点满

### 第6步：外围加成系统（有余力再堆）

| 系统 | 提升方式 | 效果 |
|---|---|---|
| 锦衣 | 锦衣升级 | 每级 +(2×级数+10) 对应属性 |
| 坐骑 | 坐骑升级（耗玩家经验）/坐骑技能升级 | 主属性×5加成+技能 |
| 称谓 | 称号修炼/突破（"破境·XX坠"提高成功率） | 防御/速度/伤害/灵力等线性叠加，冠军类称谓额外全属性1.5%-8% |
| 兽魂 | 兽魂升级（银子+经验） | 气血/速度=等级数值，35/70/105/140级解锁技能 |
| 本命法宝 | 系统经验+法宝精华升级/突破（69级解锁） | 独立属性加成 |
| 神器 | 佩戴+五行数值叠加 | 速度/法防/暴击/封印命中等 |
| 阵营 | 达成阵营任务数量 | 2%-14% 气血/速度/防御/伤害/灵力 |

- **锦衣来源**：仙玉商城锦衣分类购买、比武商店"锦衣兑换卷"、活动碎片收集（如"青花瓷锦衣·碎片"）、赞助/充值直送
- **坐骑来源**：39级"坐骑任务"（NPC太白金星，银子300万，免费获得种族坐骑）、仙玉商城祥瑞商品、30级成长礼包；坐骑升级直接消耗玩家自身经验，无独立"坐骑经验"道具
- **破境四象坠（白虎/朱雀/玄武/青龙）来源**：主要靠"破镜坠礼包""称号材料礼包"随机开出，后者可用副本积分兑换；仙玉商城曾可直购但相关代码已被注释禁用；礼包本身产出渠道代码中未找到明确来源，可能仅限活动/GM发放
- **兽魂技能领悟消耗"神兜兜"**（固定为该道具）：比武/副本积分/阵营积分/秘境积分商店均有出售，神秘宝箱也可能开出
- **法宝精华来源**：本命法宝礼包、法宝精华礼包（人族/仙族/魔族秘境积分及阵营积分兑换）、双龙签到直接发放200点；未发现怪物直接掉落
- **神器来源**：非交易道具，通过完成对应门派专属的"神器任务"副本（`Script/副本处理类/神器/`目录，每日限一次）获得神器积分，用积分+银子解锁强化，修复需"灵犀玉"

**优先级建议**：练级冲潜力上限 → 帮派点修（早期性价比最高）→ 装备成型 → 生活技能 → 锦衣/坐骑/称谓等长期堆料。

## 二、宠物属性提升攻略

**核心文件**：`Script/属性控制/宠物.lua`（属性计算 `刷新信息`）、`Script/角色处理类/召唤兽处理类.lua`（玩法入口）

### 第1步：练级 + 加点
- 喂养/经验道具升级，`加点处理()` 把潜力点分配到五维（资质×五维×成长共同决定最终属性）
- 成长系数硬顶 `min(成长, 1.5)`，所以成长超过1.5后不再吃到额外收益

### 第2步：打资质（元宵）
- 6种元宵各对应一项资质（攻/防/速/躲/体/法），每周期10次/7天，资质越高单次涨幅越小（边际递减）
- 有硬上限：普通宠1550/1550/5500/3050/1550/1800，参战等级≥155可提升到1650/1650/7000/3600/1650/2000
- **来源**：主要靠怪物/副本随机掉落（"元宵"占位符命中后随机开出六种口味之一，还可能开出如意丹/水晶糕）；另有银子商城"特殊元宵礼包"（200万银子，开出资质加成版"特殊元宵"，与打书用的六种口味是两套体系）；原NPC直购商店已被注释停用

### 第3步：进化（五彩神石，神兽向）
- 每次进化消耗递增数量五彩神石（10→60个），6项资质各+50、成长+0.05，满5次神兽→特殊神兽
- **来源**：翻一翻抽奖（仙玉付费）、嘉年华转盘抽奖、1000仙玉神秘宝箱、银子商城购买"信标"道具后击杀特殊怪物掉落、比武场地随机彩蛋（可能开出"超级"版本）

### 第4步：洗宠（资质不理想时重置）
- 金柳露（参战≤85）/超级金柳露（≥95）：普通宠资质、等级、经验清零重随，可能掉技能。**来源**：怪物/副本掉落（当前主渠道，"门派闯关""四方妖兽""长安保卫战"等表均有）、1000仙玉神秘宝箱；原NPC直购已注释停用
- 净瓶玉露：变异宠专用。**未在代码中找到当前活跃获取渠道**，可能仅限活动/GM发放
- 如意丹：微调，把已分配的加点退回1点变潜力。**来源**：NPC"宝石商人"银子购买、（侠）水陆大会等副本掉落、元宵随机池

### 第5步：打书配技能
- 技能数越少顶书成功率越高（0个技能100%，23个以上几乎顶不动），神兽类技能数上限10
- 超级技能有数量门槛（技能数≥4/6/8/12/16才能装对应个数）
- **魔兽要诀系列来源**：NPC"云游道人"银子购买、比武商店、人/仙/魔三族秘境积分商店、怪物/副本掉落（最主要渠道，"妖王""镖王""地煞星"等表均有）、仙玉商城
- 吸附石可从宠物身上拆技能做成点化石，转给其他宠物。**来源**：目前唯一确认渠道是分解装备产出（`装备处理类.lua`），非购买/掉落所得
- 法术认证可锁定关键技能不被顶替，同时随机附赠一个辅助技能

### 第6步：内丹
- 格子数随参战等级解锁：<35级1格 → 35-74级2格 → 75-104级3格 → 105-144级4格 → 145-174级5格 → ≥175级6格（神兽固定3格）
- 学内丹 → 升级内丹（消耗同名内丹叠层，上限5层）→ 开孔丸扩格
- **召唤兽内丹/高级内丹来源**：NPC"云游道人"/比武商店/副本积分商店购买、翻一翻抽奖、怪物/副本掉落、仙玉商城
- **内丹开孔丸来源**：副本积分商店、新春福利商城兑换、翻一翻抽奖
- 内丹加成 = 等级×系数×(1+内丹等级×0.2)，直接叠加到对应属性

### 第7步：装备/饰品
- 6槽装备+套装被动（3件套触发套装技能）
- 饰品直接给6项资质各+10%（乘算，性价比很高，建议优先配齐）。**圣兽丹（换饰品）来源**：NPC"宠物仙子"用"玄天残卷·上/中/下卷"（来自胡姬琵琶行、四季、无底洞、红孩儿等副本通关奖励）兑换；也可怪物/活动抽奖掉落
- 装备词条洗练可叠加百分比伤害/气血/速度/防御/灵力。**洗练石类道具来源**：翻一翻抽奖、秘境积分商店、世界事件（信标）掉落

### 第8步：进阶（灵性/特性，中后期核心提升）
- 前置：等级≥30、参战等级≥45（神兽≥60）、内丹格满且层数达标
- 易经丹推灵性到50 → 灵性≥80随机获得"特性"（力破/怒吼/瞬击等约25种，各有主效果+副作用）。**易经丹来源**：NPC"宝石商人"银子购买
- 清灵仙露继续推灵性到110（普通限7瓶/特殊直接拉满）。**来源**：初/中/高级清灵仙露主要靠拆"清灵净瓶"（NPC"宝石商人"购买）获得；"特殊清灵仙露"目前唯一确认渠道是翻一翻抽奖（稀有播报级奖品）
- 灵性满110后找NPC"御兽仙"从随机五维加成里选一项永久保留（进阶属性）
- 玉葫灵髓可重置进阶宠物五维+潜力，二次开局。**来源**：NPC"宝石商人"银子购买

### 第9步：合宠/炼妖（拼高资质宠）
- 两只宠资质取平均后按概率区间浮动生成新资质，存在上限封顶（"神话"模式更严格但资质区间更高）
- 注意：合宠本身**不消耗"炼妖石"**（只检查等级/装备/进阶状态）。"炼妖石"实际是装备锻造材料（与百炼精铁同类，按等级分档），来源为商城购买或副本/活动掉落

### 第10步：跨系统加成
- 玩家"御兽术"技能：对名下所有召唤兽全局生效（灵力/伤害/气血/速度/防御百分比），随玩家等级解锁更高级
- 坐骑"统御"技能：对被统御宠提供伤害、命中、减伤、抗性等加成（消耗坐骑饱食度）

**优先级建议**：先喂到满级+加点 → 元宵打资质到位 → 内丹格子随等级逐步补齐 → 打书成型 → 饰品配齐（乘算收益高）→ 有资源再冲进阶特性和御兽术。
