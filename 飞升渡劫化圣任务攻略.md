# 飞升 · 渡劫 · 化圣 任务攻略

> 基于服务端源码分析整理（Lua），涉及文件路径与关键函数均来自实际代码。三者是强制顺序的进阶链：**飞升 → 渡劫 → 化圣**，代码层面互为前置，无法插队或跳过。

---

## 总览：三阶段依赖链

```
主线剧情7完成(等级≥85)
      ↓ 自动切换主线=8
飞升【等级≥135，材料任务链+经验代价】
      ↓ 历劫.飞升=true，等级上限 145→155
渡劫/生死劫【对话面板155级，底层校验120级，9层顺序推图】
      ↓ 历劫.渡劫=true，等级上限 155→175
化圣【等级须恰好=175，9层，每层15亿经验】
      ↓ 历劫.化圣=true，化圣境界1→9
```

状态存放于角色对象 `self.历劫 = {飞升=false, 渡劫=false, 化圣=false}`（`Script/角色处理类/角色处理类.lua:321`），化圣下还有子字段 `历劫.化圣境界`（0~9）。

硬性前置校验（写在代码里，不是攻略建议）：
- 渡劫的 `校验个人生死劫条件` 明文要求 `历劫.飞升 == true`，否则直接拒绝。
- 角色等级上限公式：`初始上限=145`，飞升后→155，渡劫后→175（`角色处理类.lua:1941-1948`）。化圣要求等级**恰好175**，做不完渡劫根本升不到175，也就打不开化圣。

## 一、飞升（主线剧情第8章）

**触发条件**
- 等级≥135，且完成主线7（女儿国·孙悟空剧情，需等级≥85），对话入口 `Script/对话处理类/NPC对话内容.lua:2525`。
- NPC"吴刚"（地图1114）提示："135级会自动领取飞升任务"。

**任务流程**（剧情进度1→54，`NPC对话内容.lua:2525~2728`，收集+送信+战斗混合链）
1. 去明长老处受"易经洗髓"传承。
2. 捕捉"古代瑞兽""凤凰"做药引。
3. 龙宫换"镶海针"、三太子讨"开山钺"、炉神换"炼金鼎"（代价"五龙丹"+"佛光舍利子"）、黑熊精处打出"修篁斧"（战斗`110047`，等级145/气血30000/伤害2200）。
4. 集齐五大门派推荐信：剿灭"横行霸道的土匪"（战斗`110048`，等级145/气血15000）、佛门送药治瘟疫、女儿村送信（禁止飞行赶路）。
5. 方寸山"心魔"之战（`飞升_心魔1.txt`：气血199999，即玩家自身模型幻象）。
6. 魔族地藏王线：再捕"古代瑞兽/凤凰"、打"牛妖"手下（战斗`110046`）、送"回龙镊魂镖"。
7. 鬼域取"不死壤"（战斗`110049`，6个"鬼将"，气血=基础×9）。
8. **终局"五行阵法"**：点击"我准备好要破阵法了"触发战斗`110004`（`NPC对话处理.lua:2370-2372`）。怪物配置在 `怪物参数/取飞升阵法1~5战斗信息.txt`，属性**随玩家等级动态缩放**（如伤害=等级×12+800，气血=等级²×4），5套阵法轮换。

**通关结算**（`Script/战斗处理类/怪物调用/结算处理.lua:3098` `胜利MOB_110004`）
```lua
玩家数据[数字id].角色.历劫.飞升 = true
玩家数据[数字id].角色:增加剧情点(1)
-- 玉帝："想不到你居然一口气闯过了五行阵法……朕授予你三界贤者的称号。"
```

**最终确认二选一**（`NPC对话处理.lua:2373-2418`），决定是否降修炼等级：
- "我暂时保留我的修炼等级，请带我入于化境吧" → 修炼等级不降，以后可去帮派"修炼指导人"手动降修换等级上限。
- "我已经考虑好了，请降低我的修炼等级入于化境" → 直接降修炼等级。

确认后经验按当前经验分档扣除：
> ≥15亿：扣除 (当前经验-15亿)/3；5亿~15亿：清空当前经验；<5亿：扣除2亿当前经验。

**前置硬性要求**：必须有参战召唤兽（否则提示"请先将需要降级的宝宝参战"），因为召唤兽会一起执行 `飞升降级()`（等级>玩家等级+10则强制降级，按差值补偿召唤兽经验）。

| 项目 | 内容 |
|---|---|
| 触发等级 | ≥135 |
| 材料消耗 | 均为任务链内获取的专属道具（紫金冠/镶海针/开山钺/炼金鼎/修篁斧/五龙丹/佛光舍利子等），不可外部购买；另需捕捉指定宠物2次 |
| 经验代价 | 按当前经验分三档扣除（见上），唯一硬性资源惩罚 |
| 奖励 | 称谓（人"三界贤者"/仙"太乙金仙"/魔"混世魔王"）、潜力+10、师门技能重新结算、等级上限145→155 |
| 失败惩罚 | 未发现专属失败函数，按普通战斗规则处理，可重试 |
| 完成标记 | `历劫.飞升 = true` |

## 二、渡劫（生死劫九重，主线剧情第9章）

**触发条件**
- 底层校验函数 `校验个人生死劫条件`（`结算处理.lua:34-60`）：等级≥120 且 `历劫.飞升==true`，逐层前置。
- 但对话面板入口要求等级≥155（`NPC对话内容.lua:2729`）——两处数值不一致，**实际可操作门槛按155级为准**。
- 地点：凌霄宝殿（地图1112），NPC玉皇大帝。
- 组队限制：队伍**每个成员**都要单独满足条件才能一起进本，否则提示"XX+原因，无法一起进入战斗！"

**九重结构**（`结算处理.lua:6-16`，必须1→9顺序通关，不可跳跃）

| 层数 | 战斗类型 | 劫名 |
|---|---|---|
| 1~9 | 100066~100074 | 止戈/清心/雷霆/惜花/忘情/卧龙/天象/轮回/娑罗 |

**每层奖励**（`处理生死劫战斗胜利`，`结算处理.lua:248-277`）：称谓"生死劫·XX"+全队潜力+25。

**第9层额外奖励**（`处理生死劫第九劫胜利`，280-331行）：
```lua
玩家数据[cyid].角色.历劫.渡劫 = true   -- 全队同步落地
玩家数据[数字id].角色:增加剧情点(2)     -- 仅队长
同步队伍添加称谓(数字id, "生死劫·娑罗")
同步队伍增加潜力(数字id, 25)
玩家数据[数字id].道具:给予书铁(数字id, {15, 15})  -- 仅队长
```
9层通关后对话出现"步入渡劫境"，点击后剧情推进到主线10，等级上限155→175。

| 项目 | 内容 |
|---|---|
| 触发等级 | 面板155级（底层120级），且必须已飞升 |
| 材料消耗 | 无，纯战斗关卡 |
| 关卡数 | 9层，顺序完成，各层专属"劫名" |
| 每层奖励 | 称谓+潜力+25（全队） |
| 第9层奖励 | 额外称谓+潜力+25+剧情点+2（队长）+书铁×2种各15个（队长） |
| 完成标记 | `历劫.渡劫 = true` |

> ⚠️ **已知缺陷**：完成9层后进入的"主线10"复用了完全相同的9重生死劫战斗ID，但对话文本库中未找到主线10对应内容，也未找到其后续跳转逻辑，疑似未完工/预留给后续版本的二轮循环。攻略层面按"完成9层即达成渡劫"处理即可，无需理会主线10。

## 三、化圣（独立NPC任务，非主线剧情）

**NPC**："后裔英魂"（模型：进阶狂豹人形），对话编号90（`Script/对话处理类/对话调用/1001.lua:450`）。

**触发条件**（`1001.lua:2842-2846`）
```lua
if 玩家数据[数字id].角色.等级 == 175
   and 玩家数据[数字id].角色.历劫.化圣 == false
   and 玩家数据[数字id].角色.历劫.化圣境界 ~= 1 then
    战斗准备类:创建战斗(数字id+0, 120000)
```
注意等级判定是**严格等于175**（不是≥175）——因为等级上限只有渡劫完成才会提到175，所以实际效果就是"渡劫已完成且升满175级"。

**九层提升流程**：选"我要提升化圣境界（需战斗，胜利后消耗15亿经验）"，前提当前经验≥15亿，逐层触发战斗`120000`~`120008`：

```lua
function 胜利MOB_120000(...)  -- 第1层，领取任务的那一战，免经验
    历劫.化圣 = true; 历劫.化圣境界 = 1; 潜力 +30
end
function 胜利MOB_120001(...)  -- 第2层起，每层都扣15亿经验
    当前经验 -= 15亿; 历劫.化圣境界 += 1; 潜力 +30
end
-- ……120002~120007 对应境界3~8，逻辑相同
function 胜利MOB_120008(...)  -- 第9层，封顶
    历劫.化圣境界 = 9; 潜力 +30; 当前经验 -= 15亿
    if not 检查称谓("超凡入圣") then 追加称谓"超凡入圣" end
end
```

| 项目 | 内容 |
|---|---|
| 触发等级 | 恰好175（需先渡劫+练满） |
| 材料消耗 | 无道具，纯经验：第1层免费，2~9层每层扣15亿经验（共8×15亿=120亿） |
| 关卡数 | 9层，不可跳层 |
| 每层奖励 | 潜力+30（9层共+270） |
| 第9层奖励 | 额外称谓"超凡入圣" |
| 完成标记 | `历劫.化圣 = true`；`历劫.化圣境界` 累加到9封顶 |

**技能联动收益**（并非只是称号）：`Script/数据中心/技能数据库.lua` 中多个门派技能按化圣境界1/5/9三档提供递增收益——耗蓝更低、生效血量门槛更宽、增伤更高。部分门派180级师门技能在化圣后解锁额外被动（如"佛光普照"180级+化圣→"功德无量"/"醍醐灌顶"），规律见 `Script/角色处理类/角色处理类.lua` 中多处 `if self.历劫.化圣==true and jn.等级>=180`。

> ⚠️ **未实现的对话选项**："我要沐浴金乌光芒""我来看看大英雄"在代码中没有对应事件处理，点击当前版本无效果，属于占位功能。
