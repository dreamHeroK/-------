# 双龙之战活动玩法奖励说明

> 双龙之战是一套长期挂靠角色数据的"阵营玩法"（不属于《限时活动列表与奖励.md》里那种固定时刻开关的活动，也不完全是《怪物掉落和任务掉落.md》里挂机面板可勾选的明雷），本文档聚焦用户关心的两块：**小怪/BOSS掉落**、**NPC好感度的实际作用**，其余系统性内容（职位晋升、皮皮龙宠物等）仅作背景概述，不逐项展开。

## 一、数据来源

- 主体逻辑：`Script/角色处理类/简单_双龙之战.lua`（阵营数据结构 `self.数据`、加入阵营/捐赠/赠礼/职位晋升等全部交互）
- 战斗结算：`Script/战斗处理类/怪物调用/结算处理.lua`（`胜利MOB_120069`=海妖大王、`胜利MOB_120070`=深海恶蛟）、`Script/任务处理类/自写活动.lua:4430-4457`（`完成斥候海妖`）
- NPC好感度→属性加成：`Script/角色处理类/角色处理类.lua:4661-4673`（阵营达成判定）、`4802`（阵营达成初始化）、`5040-5053`（属性结算）
- 好感度获取入口：捐赠——`简单_双龙之战.lua:268-338`（NPC捐赠函数）+ `其他表/双龙NPC捐赠.txt`（运行时缓存，已用 iconv 核实当前值）；赠送礼物——`Script/对话处理类/对话调用/11181.lua`等4张NPC地图脚本（四张地图×两个阵营逻辑一致，仅NPC称谓不同）
- 道具定义：`Script/角色处理类/道具处理类.lua:16601-16620+`（`添加阵营道具`/`添加阵营礼物`）、`Script/数据中心/物品数据.lua:1910-1929`（花类道具描述原文即写明"增加NPC好感度以获得NPC祝福获得属性提升"）

## 二、玩法背景速览

- 69级解锁，二选一加入"四海龙宫"/"覆海帮"阵营（`简单_双龙之战.lua:38,944-1014`），阵营人数有上限（`特殊活动配置.ini`），满员时会自动上调上限+10并要求重新选择。
- 阵营内共7级职位（对应下文的7"页"NPC），晋升消耗历史贡献值（各级门槛1000/2000/3000/5000/8000/10000，`阵营任务达成`函数），必须按顺序逐级晋升，每级授予对应称谓并全服广播。
- 双通货：**活动积分**（消耗型，捐赠好感度/购买阵营物品/签到/训练宠物都要花）与**贡献值**（晋升型，来自捐献阵营物品、日常任务、训练宠物、击杀深海恶蛟/海妖大王等）。
- 皮皮龙宠物：加入阵营即获得，可用经验+积分+银子训练升级，等级会转化为一条可三选一切换的属性加成（伤害&法术伤害 / 防御&速度 / 封印命中&治疗&固定伤害，每级各+2点，`角色处理类.lua:4647-4659`），是另一条与NPC好感度平行的"隐藏成长线"，可通过页面21/22共享给分角色（助战）。

## 三、小怪与BOSS掉落

### 1. 斥候海妖（场景明雷，兼双龙每日任务）

- 出没：`11181`/`11191`/`11201`/`11211`/`15321` 五张地图轮询刷新，全服定时触发（也可GM/聊天指令触发），每次每张图刷30只，模型"碧水夜叉"（`自写活动:斥候海妖()`，`自写活动.lua:1713-1753`）。
- 掉落走通用HDPZ糖果产出系统：`HDPZ["斥候海妖"]` 经验倍率1、银子倍率1、**爆率100%（必掉）**，ITEM权重池：神兽涎5、修炼果5、宝石20、仙露小丸子5、仙露丸子5、九转金丹4——此条已收录于《怪物掉落和任务掉落.md》第91行，数值一致，不重复摘抄。
- 双龙专属结算（`完成斥候海妖`，`自写活动.lua:4430-4457`）：经验=平均等级×随机(220~230)×10，银子=(平均等级×160+15000)×10，储备=银子×20%，+1除妖军功；**同一只怪同时计入"每日任务[4]"进度**，当日前20只每杀1只进度+1，凑满20只即完成双龙每日任务并获得100点贡献值，无需额外操作，超过20只后不再计数也不再重复发贡献值。

### 2. 深海恶蛟（每日限战1次，长安城1001地图）

- 触发：需组队，且当日队伍内**只要有1人挑战过就整队作废**（`阵营活动数据.深海恶蛟[cyid]` 标记，`对话调用/1001.lua:863-879`）。
- 结算（`胜利MOB_120070`，`结算处理.lua:1056-1096`）：**固定**经验6000万、银子300万（不随等级浮动），随机数1~100判定1件专属道具：

  | 命中区间 | 概率 | 掉落 |
  | --- | --- | --- |
  | 1~2 | 2% | 海妖精魄 |
  | 3~4 | 2% | 超级魔兽要诀（随机词条） |
  | 5~9 | 5% | 海妖精气 |
  | 10~29 | 20% | 海妖的贡品 |
  | 30~39 | 10% | 海妖的贡品（与上一档奖励相同，属重复分支） |
  | 40~49 | 10% | 牡丹花 |
  | 50~59 | 10% | 桃运花 |
  | 60~69 | 10% | 特殊魔兽要诀 |
  | **70~100** | **31%** | 玫瑰花 |

  70~100区间已补齐为掉落玫瑰花（`结算处理.lua:1090-1092`），深海恶蛟现为100%必掉专属道具。
- 额外固定奖励：15点活动积分、50点阵营周贡献（不受上表随机结果影响，每次挑战都发）。

### 3. 海妖大王（每日限战1次，11181地图NPC）

- 与下文"好感NPC"共用同一张地图（11181的第7个假人位），需组队且当日队伍内只要有1人挑战过就整队作废（`对话调用/11181.lua:47-60`）。
- 结算（`胜利MOB_120069`，`结算处理.lua:1011-1054`）：经验=随机(40000~45000)×等级×3，银子=随机(9000~13000)×等级×3（随等级浮动），随机数1~100判定1件专属道具：

  | 命中区间 | 概率 | 掉落 |
  | --- | --- | --- |
  | 1~2 | 2% | 海妖王的灵元 |
  | 3~5 | 3% | 海妖灵兵 |
  | 6~14 | 9% | 海妖精魄 |
  | 15~49 | 35% | 海妖精气 |
  | 50~59 | 10% | 牡丹花 |
  | 60~69 | 10% | 桃运花 |
  | **70~100** | **31%** | 海妖的贡品（兜底档，事实上的"默认掉落"） |

  海妖大王每次必掉道具（100%覆盖），比深海恶蛟更稳定。
- 额外固定奖励：50点活动积分、100点贡献值、200点阵营周贡献。

## 四、NPC好感度是什么、有什么用

一句话结论：**NPC好感度不是纯剧情数值，攒满对应"页"会转化成一条永久的战斗属性加成**（气血/速度/防御/伤害/灵力同时提升），本质是双龙之战给角色提供的一条隐藏成长线——道具描述原文也直接写明"增加NPC的好感度，以获得NPC的祝福，获得属性提升"（`物品数据.lua:1911,1920,1929`）。

### 1. 好感度怎么涨

数据结构是 `self.数据.好感NPC[页][NPC序号]`，全阵营共7页、每页6个NPC，每个NPC好感度上限=**页码×100**（如第3页单个NPC封顶300）。两条互不冲突、写入同一份数据的渠道：

- **捐赠**（阵营大厅"页面8"，`NPC捐赠`函数，`简单_双龙之战.lua:268-338`）：每次消耗10点活动积分+2000万经验，任选本页某个未满的NPC类型+10点好感度；全局冷却800秒（不分NPC/页面），可花300仙玉立即清空冷却（`清空捐赠冷却`，`简单_双龙之战.lua:254-266`）。**必须按页顺序推进**：第N-1页所有6个NPC没有全部达到(N-1)×100，无法给第N页任何NPC捐赠（`NPC捐赠`函数298-315行的前置页校验）。
  - 代码细节：捐赠冷却"展示时长"和"清空所需仙玉数量"复用了同一个配置字段 `捐赠配置.清空捐献冷却`（当前值300），两者数值目前恰好相同只是巧合，如果后续想单独调整清空价格，需要先把这两处配置拆开。
- **赠送礼物**（各页对应地图当面NPC对话，如`对话调用/11181.lua`等，四张地图×两阵营逻辑一致）：消耗"牡丹花/桃运花/紫兰花/百合花/玫瑰花"5类礼物道具，分别+1/+3/+5/+7/+10点好感度，且**不受"必须按页顺序"限制**，可以跳页囤好感度。若当前阵营本周占领了该页对应地盘（`双龙之战[阵营].阵营占领[页]`），额外多送1~2点（`对话调用/11181.lua:80-83`等）。
  - 这5类花目前已验证的来源：牡丹花/桃运花——深海恶蛟与海妖大王每日挑战掉落（见上节）；紫兰花——除妖军功商店可兑换（`商业对话.lua:298-303`）；百合花/玫瑰花虽在道具表里登记为"双龙之战专属赠礼"（`物品数据.lua:1919-1929`），但未在已读代码中找到明确的获取入口，可能藏在别的商店表/活动里，暂未核实到，不排除是尚未接入产出渠道的道具。

### 2. 好感度攒满后的实际效果

- 某页6个NPC好感度全部达到"页码×100"，即视为该页"阵营达成"（`角色处理类.lua:4661-4673`）。
- 结算属性时（`角色处理类.lua:5040-5053`）取**已达成页码中的最大值 i**，加成 = i×2%，同时叠加到**最大气血、速度、防御、伤害、灵力**这5项核心属性上——**多个页面达成不叠加，只按最高页生效**，即拿到第7页的14%就不用再关心低页的2%~12%。
- 由于捐赠必须按页顺序推进，真正打到第7页满好感时前6页必然也已经满级，所以对已通关玩家而言，**14%就是这套系统实际能拿到的属性加成上限**，不会出现"只点亮低页却拿到高页加成"的情况。
- 附带功能：某页满好感后，消耗1个"朋友的礼盒"道具，可把这一页的好感度数据整体复制给一个"分角色"（助战/小号），让对方免刷直接满级该页（`共享NPC好感度`，`简单_双龙之战.lua:131-174`）。

## 五、小结 / 已发现的代码细节

1. 斥候海妖是唯一走通用HDPZ糖果系统的双龙怪，深海恶蛟/海妖大王的道具掉落都是硬编码的随机数区间判定，不受HDPZ配置影响。
2. 深海恶蛟70~100区间已调整为掉落玫瑰花，现与海妖大王一样100%必掉专属道具（末档兜底为"海妖的贡品"）。
3. 深海恶蛟掉落表里30~39区间与10~29区间奖励完全相同（都是"海妖的贡品"），像是历史调整时留下的重复分支，而非刻意设计的高权重。
4. NPC好感度的核心价值是**5项核心属性最高+14%的永久加成**，捐赠/赠礼是两条独立入口但共享同一份数据，赠礼不受页序限制、捐赠受限，规划刷好感度顺序时可以用赠礼提前囤高页进度。
5. 捐赠系统里"清空冷却展示时长"与"清空所需仙玉数"共用同一个配置字段，是潜在的配置耦合点。
6. 百合花、玫瑰花两种赠礼道具目前只确认了道具定义，未确认稳定的产出渠道，如果玩家反馈"点不出百合花/玫瑰花"，大概率是产出渠道确实缺失或藏得较深，需要进一步排查商店/活动配置表。
