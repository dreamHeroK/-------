## 一、普通特效

装备特效是"9选1/多选N"的布尔标记，没有等级/星级区分，一件装备最多携带 **1~3条**。

### 1. 候选池（按部位，`装备处理类:装备万能公式`，`装备处理类.lua:2898`）

| 部位 | 候选特效池 |
|---|---|
| 武器 | 珍宝、简易、神佑、必中、绝杀、坚固、精致、永不磨损、易修理 |
| 帽子 | 珍宝、简易、再生、坚固、精致、永不磨损、易修理 |
| 项链 | 珍宝、简易、神农、专注、坚固、精致、永不磨损、易修理 |
| 衣服 | 珍宝、简易、坚固、精致、永不磨损、易修理、伪装 |
| 腰带 | 珍宝、简易、坚固、精致、永不磨损、易修理、愤怒、暴怒 |
| 鞋子 | 珍宝、简易、坚固、精致、狩猎、迷踪、永不磨损、易修理 |

打造时（`装备处理类.lua:2967-2981`）：先抽1条必得，再各50%概率追加第2、3条，池内不重复。

### 2. 各特效实际效果（`战斗处理类:加载装备特效`，`战斗处理.lua:2925-2947`）

| 特效 | 效果 |
|---|---|
| 愤怒 | 主动技能/特技的愤怒消耗降为80% |
| 暴怒 | 每次涨怒气额外+20% |
| 神佑 | 受致命伤时20%概率免死，回血至最大气血60% |
| 必中 | 普攻/固定伤害技能永不因命中<躲避而落空 |
| 简易 | 装备等级要求降低5级（角色等级+5≥装备等级即可穿） |
| 精致 | 额外+1个宝石镶嵌孔 |
| 无级别限制 | 完全无视装备等级要求，且解除宝石镶嵌孔上限（见第三节） |
| 超级简易 | 效果同"简易"，但存储为字符串而非表，来源见第三节 |

| 珍宝、坚固、易修理、专注、伪装、狩猎、迷踪 | **未找到功能实现**，属于占位/预留特效 |

### 3. 特技（39~41种，视口径而定）

权威登记表 `Script\数据中心\装备特技.lua`，共46个技能名，用于战斗中标记"这是特技触发"。三个不同的随机来源口径不同：

- **普通打造随机池**：40个（`装备处理类.lua:2984`），比登记表少 `琴音三叠、移行换影、虚弱、易伤、停陷术、死亡之音`。
- **160级神兵苏醒随机池**：41个（`装备处理类.lua:3337-3339`），比普通池多了 `琴音三叠`。
- **"特技书"重随池**：41个去重后唯一名（`道具处理类.lua:14933`），但其中 `气归术/凝气诀/凝神诀/命疗术/流云诀/啸风诀/河东狮吼/破甲术` 各重复写了两遍，被抽中权重是其余33个的2倍（实现缺陷，见第五节）。

`移行换影、虚弱、易伤、停陷术、死亡之音` 这5个特技虽有完整战斗实现，但不在上述任何随机来源中，只能靠策划定制装备直接赋值获得。

特技效果说明见 `Script\数据中心\技能数据库.lua:5788-6335`（字段 `是否特技=true`），按类型大致分为：物攻类（破血狂攻、破碎无双）、恢复类（命归术/气归术/命疗术/心疗术/气疗术/水清诀/玉清诀/冰清诀/晶清诀/四海升平/慈航普度/起死回生/回魂咒/凝气诀/凝神诀/金刚怒目）、固伤/减益类（诅咒之伤、吸血特技、放下屠刀、笑里藏刀、碎甲术/破甲术、河东狮吼、凝滞术/停陷术）、增益类（太极护法/罗汉金钟、心如明镜、流云诀/啸风诀、野兽之力/魔兽之印、光辉之甲/圣灵之甲、金刚不坏/菩提心佑、修罗咒/身似菩提）。

## 二、打造时获得特效/特技的概率

### 1. 普通打造（`装备处理类:装备万能公式`，`装备处理类.lua:2898-3048`）
- 特效：`取随机数()<=5` 即 **5%概率**（2967-2981行）
- 特技：`取随机数()<=4` 即 **4%概率**，40选1（2982-2986行）

### 2. 160级"神兵苏醒"强化打造（`装备处理类:取160装备属性`+`打造160神兵`，`装备处理类.lua:3175-3427`）
概率随生活技能熟练度提升，与普通打造是独立的第二套公式：
- `SLD = 取随机数(1, 2+min(ceil(熟练度/200),5))`（3212-3218行）
- 命中判定：`特效 <= floor(SLD/2) + 特效概率累加值`（3318-3323行），特效概率累加值来自元身属性历次强化的累积
- 命中特效后，10%概率触发"双特效"，同时获得传统池1条 + 独立的"新特效"（`道具.新特效`）；否则50/50二选一
- "新特效"候选池（3340行）：物理暴击几率、法术暴击几率、物理暴击伤害、法术暴击伤害、治疗能力、封印命中率、抵抗封印命中率、穿刺效果、格挡物理伤害、法术伤害减免、魔法回复效果——专属160级神兵，独立于普通9特效
- 特技命中后固定41选1

### 3. 幻化装备（`装备处理类:幻化公式`，`装备处理类.lua:3112-3173`）
另一独立公式，各自5%特效/5%特技判定（3164-3171行），基础属性范围不同于普通打造，不要与前两者混淆。

## 三、鉴定获得"蓝字特效（无级别限制）"

核心函数：`装备处理类:装备鉴定处理`，`装备处理类.lua:2306-2401`；调用入口为 `道具处理类.lua:14518`（普通鉴定）与 `道具处理类.lua:15502`（专用装备鉴定）。

判定逻辑（2306-2335行）：
```
无级别概率 = 1   -- 基础1%
if 神话 and 装备属性.制造者 then
    等级120~130 → 80%
    等级140~149 → 30%
    等级150~159 → 15%
    等级≥160    → 3%
end
```

> **实测结论**：全局开关 `神话` 在当前代码中未被任何位置赋值为 `true`（只在几十处被读取判断），恒为假。因此上述 80%/30%/15%/3% 梯度是**死代码，永远不会命中**，鉴定"无级别限制"当前实际生效概率恒定为 **1%**。文档/运营如需启用该梯度，需要先找到并打开 `神话` 开关的赋值逻辑。

装备等级≥80时，鉴定成功会根据已有特效/特技数量弹出"双/三/四蓝字"提示并做全服广播（2372-2400行），这只是展示逻辑，不产生额外概率。

## 四、存储字段与战斗生效链路

- `道具.特效`：数组，如 `{"珍宝","坚固","无级别限制"}`
- `道具.特技`：单个字符串，如 `"凝气诀"`
- `道具.新特效`/`道具.新特效数值`：160级神兵专属数值型效果

汇总链路：
1. `角色处理类:取特效()`（`角色处理类.lua:6260-6270`）、`角色处理类:取特技()`（`6242-6258`）——遍历6个装备槽收集去重
2. 战斗初始化（`战斗处理.lua:3240-3244`）：特技作为等级10的主动技能加入 `参战单位.特技技能`，与其他学习技能同流程结算；特效写入 `加载装备特效`（`战斗处理.lua:2925-2947`）对应字段
3. 160级"新特效"结算：`战斗处理类:计算160属性`（`战斗处理.lua:2965-3010`）

## 六、已知实现问题（供文档撰写/后续开发参考，避免误导玩家）

1. **"绝杀""神农""再生"仅被赋值，从未被任何战斗计算代码读取**，打造出来只是好看，无数值效果。
2. **"珍宝""专注""伪装""狩猎""迷踪""易修理"未找到任何功能实现**，属于占位/预留特效。
3. **"永不磨损"存在类型不匹配Bug**：`扣除装备耐久`函数（`道具处理类.lua:13639,13669`）用 `.特效 ~= "永不磨损"` 做字符串比较，但 `.特效` 字段实际是table数组，比较恒为真，导致"永不磨损"对耐久消耗**不生效**。
4. **"特技书"重随池权重Bug**：8个特技名被重复写了两遍，被抽中概率是其余33个的2倍（`道具处理类.lua:14933`）。
5. **"神话"全局开关未启用**：导致鉴定"无级别限制"按等级80%/30%/15%/3%浮动的设计从未生效，当前恒定1%（详见第三节）。
6. **移行换影、虚弱、易伤、停陷术、死亡之音**：5个特技有完整战斗实现，但不在任何随机获取途径中，只能靠策划定制装备获得。
